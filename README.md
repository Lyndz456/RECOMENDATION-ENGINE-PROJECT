#A FINANCIAL RECOMENDATION-ENGINE-PROJECT
# Data-Driven Financial Recommendation System for Kenyan Income Earners

## Project Overview

This project develops a machine learning-based financial recommendation system that helps income earners in Kenya make better financial decisions by providing personalised recommendations for suitable financial products.

The system creates customer profiles, segments users into behavioural clusters using unsupervised learning (K-Means or Hierarchical Clustering), and recommends appropriate financial instruments including Money Market Funds, SACCO savings accounts, and securities exchange investments.

## Research Problem

Despite 84.8% of Kenyan adults having access to formal financial services, financial health has collapsed from 39.4% in 2016 to just 18.3% in 2024. While 42.1% of Kenyans demonstrate high financial literacy, only 36% save formally. This literacy-action gap is the problem this system addresses.
